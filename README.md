# PG-Pallium_paper

This repository contains all of the MATLAB codes used for the analysis of the 2-P neural data and used for creating the figures in the manuscript titled: "Hierarchical sensory processing in zebrafish
thalamocortical-like circuits" (Trinh et al., 2026) [DOI:](https://doi.org/10.1126/science.aec2171)
Each Figure has it's own matlab script and the implemented functions are collected in the "Subfunction" folder. 

## Requirements: 
#### All code are written in MATLAB and was tested on MATLAB versions 2024a and 2025a
#### The dataset containing the processed neural activity recorded using a 2-photon Calcium Imaging can be found at: https://doi.org/10.11582/2026.gvso70gj
#### The code runs on the standard MATLAB version (https://www.mathworks.com/products/matlab.html) please follow the instruction to install it there.
#### Any extra function used in specific scripts are grouped together in the "Subfunction" folder.

-	The data path (plus other required paths) needs to be replaced for your system and then the code must be run per section in the order that it was written.
